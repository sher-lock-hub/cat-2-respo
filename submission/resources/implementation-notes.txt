STEP SIX:Local storage
So for this it's the wish lis
So when someone refreshes the page the wishlist is still there
saveWishlist() is a function that saves the current wishlist to the browser's localStorage. Whenever a user adds or removes a jewellery item, the function collects all the items currently displayed in the wishlist and stores them so that they are not lost when the page is refreshed.

loadWishlist() is a function that retrieves the saved wishlist from localStorage when the page loads. It reads the previously stored items and recreates them in the wishlist so that the user can continue where they left off, even after closing or refreshing the browser.
